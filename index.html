My Recent Work

dREaMEMBER

dREaMEMBER is a dream analysis and journaling application developed with Express.js, AngularJS, Node.js, MongoDB, JQuery, Tubular.js, Bcrypt, HTML5, CSS3, and Bootstrap.

Concrt

This concert finding and sharing app was built using Ruby on Rails and utilizes HTML5, CSS3, Bootstrap, Bootstrap Datepicker, D3.js, Cloudinary, JQuery Panelsnap, Bcrypt, and PostgreSQL.

Holidaze

Holidaze was the product of a day-long hackathon project during my WDI course at General Assembly. This application, built with React, allows a user to search for and add new holiday recipes to the database.

Karaoke-Dokie

Karaoke-Dokie is an application for finding Seattle's best karaoke spots. Built on Express and Node, it incorporates both the Google Maps and Yelp APIs, and utilizes Facebook and Passport.js for user authentication.
